## 常见问题
### 1、deepin chrome 代理出现问题
解决：[deepin chrome代理出现问题 mangoogle-chrome-stable 问题解决](https://blog.csdn.net/qq_42443261/article/details/90737973)

### 2、默认资源管理器变成了 vscode
解决：双击打开资源管理器 -> 选择一个文件 -> 右键选择打开方式 -> 默认程序勾选文件管理器

### 3、软件窗口顶部大边框问题
